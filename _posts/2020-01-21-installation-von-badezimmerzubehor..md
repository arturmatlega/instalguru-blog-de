## Wie wird ein Becken ausgelotet?

Es gibt mehrere Methoden für die Installation von Becken. Eine wandhängende Schüssel wird mit Bolzen und Halterungen an der Wand befestigt. Es gibt auch stehende Spülbecken in Kombination mit Schränken - Waschtischunterschränken.

{:.distinction.distinction-info.distinction-negative}

Abhängig von einem bestimmten Modell wird der Installationsprozess unterschiedlich verlaufen.

### Wandhängendes Becken

Diese Art von Badezimmermöbeln haben in der Regel runde oder quadratische Formen. Wie schwierig es ist, ein Waschbecken an der Wand zu befestigen, hängt von seiner Form ab. Denken Sie daran, dass bei dieser Art von Geräten ein Geruchsverschluss sichtbar bleibt. Aus diesem Grund ist es eine gute Idee, ein Gerät zu wählen, das zum Rest des gesamten Baddesigns passt. Ein Chromelement könnte in diesem Fall die beste Lösung sein.

{:.distinction.distinction-quotation.distinction-positive}

Es ist auch möglich, die Installationselemente hinter einem Sockel oder einer Siphonabdeckung zu verstecken. Dies ist in diesem Fall eine der beliebtesten Lösungen.

Beginnen Sie die Installation damit, einen automatischen Spülbeckenabfluss in den Abfluss zu stellen. Installieren Sie dann den Wasserhahn. Denken Sie daran, Dichtungen einzusetzen, die das Spülbecken vor dem Auslaufen schützen. Bringen Sie den Geruchsverschluss an und setzen Sie ein Abflussrohr (denken Sie daran, eine Dichtung anzubringen) und eine Kappe ein. Sie können das von Hand machen, Spezialwerkzeuge sind in diesem Fall nicht erforderlich.

### Eitelkeits-Einheit

Dieser Typ enthält in der Regel ein Becken in Form eines Ovals oder Kreises. Die Installation sollte damit beginnen, zwei Löcher in den Schrank zu schneiden - für einen Wasserhahn und für einen Abfluss. Dann setzen Sie die Schale in die richtige Öffnung. Befestigen Sie den Boden des Waschbeckens mit transparentem Silikon am Schrank. Verteilen Sie die Substanz an den Rändern und drücken Sie das Becken gegen den Schrank. Wischen Sie das überschüssige Silikon mit einem weichen, feuchten Tuch ab. Der letzte Schritt umfasst die Installation des Geruchsverschlusses und den Anschluss der Geräte an das Wassersystem. Verwenden Sie dazu flexible Schläuche oder Kupferrohre.

{:.distinction.distinction-info.distinction-negative}

Denken Sie daran, vor Beginn der Installation die Hauptwasserversorgung abzuschalten.

## Installieren eines Bades

Wo soll ein Bad installiert werden? Die besten und am häufigsten gewählten Stellen für den Einbau sind Badezimmerwände. Achten Sie auf den Boden - der Boden sollte eben sein, damit die Badewanne stabil ist. Wenn Sie sich entschieden haben, eine Badewanne in Ihrem Badezimmer zu installieren, sollten Sie von den Beinen ausgehen - befestigen Sie sie an der Wanne und regulieren Sie ihre Höhe.

Wenn Sie sicher sind, daß die Badewanne richtig eingepaßt ist, ziehen Sie die Beine fest an, so daß sie stabil sind. Der nächste Schritt besteht darin, den Siphon und das Abflussrohr an der geeigneten Stelle anzubringen. Denken Sie daran, an beiden Elementen Dichtungen anzubringen, damit das neu geschaffene System nicht undicht wird. Als letztes muss der Badeschlauch an der speziellen Ablauföffnung befestigt werden.

{:.distinction.distinction-quotation.distinction-positive}

Nachdem Sie das gesamte System installiert haben, bringen Sie Silikon auf die Leisten auf und sichern Sie das Bad.

## Wie installiert man eine Toilette?

Bei einer Toiletteninstallation muss ein Spülmechanismus in die größte Öffnung eingesetzt werden. Denken Sie daran, eine Dichtung anzubringen. Die beiden verbleibenden Löcher sind für Bolzen und Kappen vorgesehen. Die Spülung sollte im Tank positioniert und alle Teile mit Polypropylenmuttern gesichert werden. Das gesamte Gerät muss stabil sein. Achten Sie darauf, dass die Spülung nicht verrutscht.

Der nächste Schritt ist das Einsetzen einer Schwimmerkugel. Sie sollte links oder rechts von der Spülung platziert werden. Setzen Sie den Deckel auf und führen Sie das Spülventil in das Loch im Inneren ein.

Stecken Sie ein versetztes Gummi-Pfannenverbindungsstück in das Abflussrohr, das sich im Badezimmerboden befindet. Es muss mit dem senkrechten Abfluss der Toilettenschüssel verbunden werden. Markieren Sie Stellen für Schrauben. Stellen Sie die Schüssel auf den Boden und richten Sie sie an den Punkten für die Schrauben aus. Schrauben Sie die Bolzen ein und setzen Sie Dübel ein. Richten Sie die Toilettenschüssel wieder aus und schrauben Sie sie mit einem Steckschlüssel fest.

{:.distinction.distinction-info.distinction-positive}

Achten Sie darauf, dass die Toilette fixiert ist und sich nicht bewegt.

Setzen Sie Kappen auf die Bolzen. Die Spülung sollte mit Toilettenzubehör - in diesem Fall einem flexiblen Schlauch - am Ventil befestigt werden. Um ein besseres Ergebnis zu erzielen, decken Sie den Anschluss mit Silikon ab.

Die Installation von Toilettenzubehör ist nur scheinbar schwierig. Dank entsprechender Handbücher kann man sie selbst durchführen.
